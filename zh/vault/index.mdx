Proportion 的核心是 Vault contract。它协调 LP deposits、funded-account financing、USDC custody、fees 和 profit distribution。协议中的每一笔 capital flow 都经过 Vault。

## Shares and sharePrice

LP 在 Vault 中的 position 由 shares 表示。累计 profit 会提高 `sharePrice`；yield 不需要单独 claim。

```
sharePrice = totalAssets / totalShares
totalAssets = USDC on EVM + USDC allocated to active FAs
```

## Allocation Limits

| Parameter | Value | What it limits |
|-----------|-------|----------------|
| Max utilization | **80%** | 分配给 active FAs 的 Vault 份额 |
| Max single account | **10%** / $20k | 单个 FA 相对 `totalAssets` 的规模 |
| Builder leverage | **10x** | Builder funding 相对 `sharesValue` 的最大倍数 |

## Lock-Up

没有 Builder role 的 LP 可以随时 withdraw shares。Builder shares 在其拥有 active FAs 时会被锁定。`withdrawableShares()` 显示可 withdrawal 的 shares 数量。

## Governance

Vault parameters（`feeMultiplier`、`MIN/MAX_ACCOUNT_SIZE`、`MIN/MAX_DRAWDOWN_BPS`、`BUILDER_LEVERAGE`、utilization、per-FA limits）由 admin 通过 setters 修改。
